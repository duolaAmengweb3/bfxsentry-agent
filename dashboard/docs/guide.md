# BfxSentry — 从 Bitfinex 鲸鱼数据中提取交易信号

> 监控 Bitfinex 市场中真正有意义的数据：谁在赚钱、谁在爆仓、谁在挂大单、资金成本多少。
> 把这些信息变成可执行的交易信号，自动化运行。

---

## 一句话解释

大多数交易者盯着 K 线和技术指标做决策，但真正影响价格的是**鲸鱼的动作**——他们的仓位变化、爆仓清算、大额挂单。

BfxSentry 做的事情很简单：**从 Bitfinex 公开数据中找到鲸鱼的痕迹，判断他们的方向，然后跟着走**。

为什么是 Bitfinex？因为它是目前主流交易所中**唯一仍公开完整交易者排行榜**的平台。你可以看到 Top 100 交易者的盈利数据，这在 Binance、OKX 上是看不到的。

---

## 它能看到什么

BfxSentry 从 Bitfinex 公开 API 中提取四类关键数据，这些数据不需要登录、不需要 API Key，任何人都可以获取：

### 鲸鱼持仓方向

Bitfinex 公开所有交易者的盈利排行榜。BfxSentry 分析 Top 100 交易者的 PnL 数据，通过数学方法（Pearson 相关性）推算出他们当前**做多还是做空**。

当多数鲸鱼方向一致时，这就是一个强信号。

### 爆仓清算流

每一笔强制平仓记录都带有时间、方向、金额。BfxSentry 实时统计爆仓强度，判断是否出现了**级联爆仓**——大量多头或空头被连环清算。

这通常意味着短期价格过度偏离，接下来可能反弹，也可能继续加速。判断取决于爆仓的密度和市场的反应。

### 盘口深度

完整的 25 档买卖盘口数据。BfxSentry 检测两种关键情况：

- **深度失衡** — 买盘远厚于卖盘（或反过来），暗示大资金的方向倾向
- **挂单墙** — 某个价位出现异常大的挂单量，通常代表一个关键支撑/阻力位

### 融资利率

Bitfinex 有一个独特的融资市场。交易者借入 USD 加杠杆时需要付利息，利率由市场供需决定。

当利率飙升且利用率超过 80%，说明市场杠杆很重，波动概率增加。同时也意味着放贷的年化收益可能超过 30%。

## 五个策略，各有分工

BfxSentry 不是一个单一策略的交易机器人。它包含五个独立的策略模块，各自关注不同维度的数据，做不同类型的交易。

### 聪明钱跟单

**逻辑**：当排行榜上 5 位以上的顶尖交易者同方向持仓，且综合评分超过 65 时，跟随他们的方向开仓。

**特点**：中低频交易，每 3 分钟评估一次。适合趋势行情，不追求精确入场点，追求方向正确。

### 融资套利

**逻辑**：不猜方向，纯粹赚利息。当融资利率处于历史高位且借贷需求旺盛时，将资金放出去赚取利息。

**特点**：零方向性风险。牛市高杠杆时期，年化利率经常达到 30-100%。这是 Bitfinex 独有的机会——2025 年 12 月起 Bitfinex 全品种 0 手续费。

### 爆仓猎人

**逻辑**：监控实时爆仓记录。当爆仓强度达到历史 P85（超过 85% 的正常时段），且单边爆仓占比超过 70% 时触发。

有两种打法：
- **顺势**：爆仓方向与趋势一致，价格还在加速，跟着走
- **逆势**：极端爆仓（P95+）但价格停滞，博弈反弹

**特点**：高频评估（5 秒），严格止损。只在极端行情中出手。

### 盘口狙击

**逻辑**：当买卖盘深度出现 1.5 倍以上失衡，且最近 60 秒的成交方向确认了失衡方向时，短线入场。

**特点**：最高频的策略，止损极紧（0.3%），止盈同样不贪（0.5%）。纯粹基于微观结构，与宏观趋势无关。

### Polymarket 信号下注

**逻辑**：将上述四个模块的信号聚合，计算综合置信度。当置信度超过 35% 时，在 Polymarket 的 BTC 短周期预测市场买入对应方向。

**为什么 35%？** 30 天回测验证：

| 置信度门槛 | 信号次数 | 预测准确率 | 预期收益 |
|-----------|---------|-----------|---------|
| ≥ 30% | 17 次 | 52.9% | +3.9% |
| ≥ 35% | 9 次 | 55.6% | +9.1% |
| ≥ 40% | 6 次 | 66.7% | +31.3% |

35% 是信号频率和准确率的最佳平衡点。太低则噪声太多，太高则信号太少。

**关键设计**：单模块信号不会触发——必须多个模块同时给出方向性判断才会下注。这避免了单一数据源的误判。

## 风控不是口号

每个策略有独立的仓位限额、杠杆上限和止损参数，互不干扰。策略之上还有一层组合级风控：

- 所有策略加起来总仓位不超过 $5,000
- 单日亏损达到 $200 即锁定
- 最多同时持有 3 个仓位
- 同方向集中度不超过 70%

**硬性熔断**：连续 5 次止损，或累计亏损达 $300，自动暂停全部策略 60 分钟。没有例外，没有手动覆盖。

所有参数可调，但保护机制不能关闭。

## 从安装到第一次扫描，3 分钟

```bash
npm install -g bfxsentry-agent
```

装完就可以用。**所有只读功能不需要任何 API Key**——扫描、信号查看、策略状态、回测全部免费直接使用。

```bash
sentry scan
```

这一条命令会同时拉取 Bitfinex 四个维度的数据，输出：鲸鱼方向及置信度、爆仓强度、盘口失衡程度、融资利率状态。

```bash
# 查看某个维度的详细信号
sentry signal smart-money
sentry signal liquidation
sentry signal orderbook
sentry signal funding

# 查看策略状态
sentry strategy
```

### 回测验证

不信？跑一下历史数据看看：

```bash
sentry backtest -d 7
```

回测引擎直接从 Bitfinex API 拉取历史数据，不需要任何预录文件。它会自动测试 225 种参数组合，找到最优配置，输出胜率、收益、最大回撤等关键指标。

### 模拟运行

```bash
sentry start
```

默认是 dry-run 模式——信号正常触发，但不会真正下单。所有决策记录在本地数据库，可以随时查看。

### 实盘交易

确认策略有效后，切换到实盘：

```bash
export BFX_API_KEY=你的key
export BFX_API_SECRET=你的secret
sentry start --mode live
```

Bitfinex API Key 在 bitfinex.com 后台生成，只需开启交易权限。

如果还需要 Polymarket 自动下注，额外安装官方 CLI：

```bash
brew install polymarket-cli
polymarket auth           # 连接钱包
sentry start --mode live  # BfxSentry 会在信号达标时自动调用
```

## Web Dashboard — 实时可视化

除了命令行工具，BfxSentry 还有一个 Web 仪表盘，把所有数据实时可视化呈现：

| 模块 | 你能看到什么 |
|------|------------|
| 数据中心 | BTC 价格、仓位变化、成交量、费率的综合概览 |
| 交易龙虎榜 | Bitfinex Top 100 交易者实时盈利排名和持仓变化 |
| 聪明钱信号 | 鲸鱼方向推断、历史准确率、信号强度走势 |
| 融资雷达 | FRR 利率走势、利用率、放贷最佳时机提示 |
| 爆仓猎人 | 实时爆仓记录流、强度百分位、级联检测 |
| 盘口狙击 | 25 档深度热力图、失衡比率、大单墙标记 |
| PM 雷达 | Polymarket BTC 市场赔率、信号下注记录、回测准确率 |

Dashboard 所有数据来自 Bitfinex 公开 API，2-10 秒刷新。不需要任何注册或认证。

## 谁适合用

- **主观交易者**：不用它来自动交易，只用 `sentry scan` 看信号，辅助自己的判断
- **量化交易者**：直接用五个策略模块自动运行，关注回测结果和参数优化
- **利率套利者**：只开 funding-arb 策略，专注 Bitfinex 融资市场的利率收益
- **Polymarket 玩家**：用四维信号聚合做 BTC 短周期方向预测，比拍脑袋靠谱
- **数据研究者**：用 Dashboard 和 `sentry signal` 追踪 Bitfinex 市场微观结构

export SOCKS_PROXY=socks5h://127.0.0.1:7897

## 核心优势总结

**数据独占性** — Bitfinex 是唯一公开完整交易者排行榜的主流交易所。别人看不到的数据，你能看到。

**零成本验证** — `npm install` 之后立刻 `sentry scan`，不需要 Key、不需要注册、不花钱。先看信号质量，再决定是否投入。

**回测即证据** — 所有策略的表现都可以用历史数据验证，225 种参数组合自动寻优。不靠信仰，靠数据说话。

**0 手续费执行** — Bitfinex 2025 年 12 月起全品种 maker + taker 均为 0 费率，策略执行零摩擦。

**多维度聚合** — 单一信号容易误判。BfxSentry 聚合鲸鱼持仓、爆仓流、盘口深度、融资利率四个维度，多模块共识才触发交易。
